* ENV Setup with conda:

** Activate base anaconda:
#+begin_src sh :shebang #!/bin/sh :results output :tangle ./shrc_install.sh
  . /opt/anaconda/bin/activate
#+end_src

** Create env:
#+begin_src sh :shebang #!/bin/sh :results output :tangle ./shrc_install.sh
  conda create -n myenv
#+end_src

** Activate env:
#+begin_src sh :shebang #!/bin/sh :results output :tangle ./shrc_install.sh
  conda activate myenv
#+end_src
